# Code Alpha Calculator

## Description
A simple web-based calculator built with **HTML, CSS, and JavaScript**, developed during my internship at **Code Alpha**.  
It supports basic arithmetic operations, percentage, and square root calculations.  

<img width="519" height="580" alt="image" src="https://github.com/user-attachments/assets/5c95aa93-f0a3-4e8b-9a04-c8446182e568" />

## Features
- Addition, Subtraction, Multiplication, Division
- Percentage calculation
- Square root and square functionality
- Keyboard input support
- Clear all button functionality
